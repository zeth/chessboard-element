Position FEN

You can set the position property to a FEN string.

HTML

<chess-board
    position="r1bqkbnr/pppp1ppp/2n5/1B2p3/4P3/5N2/PPPP1PPP/RNBQK2R"
    style="width: 400px">
</chess-board>
